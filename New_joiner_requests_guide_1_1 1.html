Common issues and solutions post software tool installations :

1.Python Package Repository:

URL: https://files.pythonhosted.org/

Purpose: Required for installing Python libraries via pip.

solution : Raise an access to restricted website request with Barred url https://files.pythonhosted.org/ and attach error screenshot. Once approved the pip installer works.

2nd solution : Raise a request for Disney – Admin Access through the OneConnect portal. Search for the catalog item “Admin Access” and complete all required fields appropriately. Once access is granted, configure the required proxy settings by adding the following System Environment Variables under System Variables → New:
HTTP_PROXY = http://ep.threatpulse.net:80
HTTPS_PROXY = http://ep.threatpulse.net:80
After configuring the variables, restart the system to ensure the changes take effect.

2.Node.js Package Registry:

URL: https://registry.npmjs.org/

Purpose: Required for installing Node.js packages via npm.

solution : Raise an access to restricted website request with Barred url https://registry.npmjs.org/ and attach error screenshot. Once approved the npm install should work.

3.Vscode extensions like copilot,pylance, httpclient etc.

solution : Raise an Incident with the description vs code extensions not working and IT support will reset the required permissions and extentions will start to work.

2nd solution : If the extension installation issue persists even after raising a request with the IT Support team, temporarily connect your laptop to a personal hotspot or external Wi‑Fi network (if permitted by policy). Download the required VS Code extension manually in .vsix format. Then follow these steps:
1. Open VS Code
2. Navigate to Extensions
3. Click on the three dots (⋯) at the top-right
4. Select Install from VSIX…
5. Choose the downloaded file and complete the installation
Restart VS Code after installation if required.

4. Installing WSL

Problem/error while installing :A specified logon session does not exist. It may already have been terminated..

solution: Connect with IT team to get admin credentials, Turn On windows features for Windows SubSystem for Linux.

Use https://learn.microsoft.com/en-us/windows/wsl/install-manual#step-4---download-the-linux-kernel-update-package to upgrade WSL kernel.
